Proposito

Este sitio existe para facilitar el trabajo de tésis o trabajo profesional a los alumnos de Ingeniería en Informática de la Facultad de Ingenieria de la Universidad de Buenos Aires (FIUBA)

Si desea contribuir, puede hacerlo abriendo un issue detallando la cuestión, o mejor aún, haciendo un fork y pull request con los cambios que desea.

El respositorio se encuentra en https://github.com/infotytpsistfiuba/infotytpsistfiuba.github.io.

El readme contiene instrucciones técnicas más detalladas.
